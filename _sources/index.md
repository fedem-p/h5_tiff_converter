
# File converter from h5 to tiff images

[![Documentation](https://github.com/fedem-p/h5_tiff_converter/actions/workflows/page_deploy.yml/badge.svg)](https://fedem-p.github.io/h5_tiff_converter/)
[![pypi](https://badge.fury.io/py/h5tiff.svg)](https://pypi.org/project/h5tiff/)
Check the repository [here](https://github.com/fedem-p/h5_tiff_converter)

A simple package to convert a list of images saved in a h5 file to single tiff files.

The package is structured by a class which holds the data and metadata (with customizable loading function).
The class has a method call convert which will convert and save the images. (also customizable)
